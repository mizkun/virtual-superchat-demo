# virtual-superchat-demo

Youtube APIを用いて、Youtube Liveのメッセージや、SuperChatで貰った金額に対応するVirtual GiftをVR空間に出現させるデモです。
かなり雑なので、雰囲気を掴んでもらう参考になれば。

UnityからYoutubeへのOAuth認証周りは、下記のブログを参考にさせて頂きました。

https://qiita.com/n0mimono/items/0c2921c9b964cd305fe3

Youtube Live Streaming APIの仕様については、下記のページにすべてまとまっています。

https://developers.google.com/youtube/v3/live/docs/
